## Installation

Requires Go 1.20+

```bash
go install github.com/kostovJP/taskcli@latest
```
Make sure $(go env GOPATH)/bin is in your PATH.

To check, run: 

if you use bash:
```bash
echo 'export PATH=$PATH:$(go env GOPATH)/bin' >> ~/.bashrc
```

if you use zsh:
```bash
echo 'export PATH=$PATH:$(go env GOPATH)/bin' >> ~/.zshrc
```

Then reload:
```bash
source ~/.bashrc
```
or
```bash
source ~/.zshrc
```
